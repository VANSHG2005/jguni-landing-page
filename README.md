# JG University — Landing Page

A modern, responsive university landing page built with **React + Tailwind CSS**, inspired by [jguni.in](https://jguni.in/).

## Tech Stack
- React 18 (Vite)
- Tailwind CSS v3
- Google Fonts: Cormorant Garamond + DM Sans

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build for Production

```bash
npm run build
npm run preview
```

## Sections
1. **Navbar** — sticky, scroll-reactive with gold hover effects
2. **Hero** — full-viewport with animated rings, stats strip, CTAs
3. **Marquee** — programme ticker ribbon
4. **About** — two-column layout with feature cards
5. **Programs** — filterable grid (Business / Technology / Commerce)
6. **Testimonials** — student quote cards
7. **CTA** — admissions call-to-action
8. **Footer** — full links, social icons, legal

## Design System
- Background: `#050c1a` (deep navy)
- Accent: `#c9a84c` (gold)
- Display font: Cormorant Garamond (serif)
- Body font: DM Sans
